# CultureCare 정책 처방지도

문화데이터 기반 전국 시군구 문화접근 취약도 진단 플랫폼 GitHub Pages 템플릿입니다.

## 사용 방법

1. 이 폴더 전체를 GitHub 저장소에 업로드합니다.
2. `assets` 폴더에 아래 그림 파일을 넣습니다.
   - figure_17_national_culturecare_vulnerability_map.png
   - figure_19_lisa_cluster_map.png
   - figure_23_spatial_priority_target_map.png
   - figure_14_top20_targeted_policy_effect_heatmap.png
   - figure_15_best_policy_by_high_risk_region.png
   - figure_16_balanced_policy_before_after.png
3. GitHub 저장소 Settings → Pages → Branch를 `main` / root로 설정합니다.
4. 생성된 GitHub Pages 링크를 제출자료에 첨부합니다.

## 수정할 파일

- `index.html`: 웹페이지 문구와 섹션 구조
- `style.css`: 디자인
- `script.js`: 지역 검색 및 진단 카드 데이터
- `data/top20_regions.csv`: 상위 20개 지역 데이터
- `data/priority12_regions.csv`: 최우선 공간정책 대상 12개 지역
